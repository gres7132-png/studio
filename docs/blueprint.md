# **App Name**: Balenciaga Marketing Rights

## Core Features:

- Dashboard Metrics: Display account balance, today's earnings, recharge amount, and withdrawal amount on the dashboard.
- Marketing Rights Packages: Display packages as 'Purchase of Balenciaga product marketing rights'. Show product images of Balenciaga-branded t-shirts alongside each package.
- Agent Commissions: Provide an agent commission tracking page.
- Payment Verification: On successful M-Pesa payment, include a field for pasting the payment confirmation message for admin verification.
- Cryptocurrency Payments: Allow payments via cryptocurrency with a wallet address displayed for payment. Details such as coin type will be handled during the setup.
- Package Configuration: Enable admin to change product marketing rights packages via the admin panel.
- Promotional Offers: Enable admin to add limited-time promotional offers on investment packages via the admin panel.

## Style Guidelines:

- Primary color: Deep Blue (#1A237E) for a sophisticated and trustworthy feel, reflecting the luxury brand.
- Background color: Very light gray (#F5F5F5) to provide a clean, modern canvas for the content.
- Accent color: Gold (#FFD700) to highlight important elements and convey premium quality and elegance.
- Font pairing: 'Belleza' (sans-serif) for headlines, communicating a sense of fashion, art and design, matched with 'Alegreya' (serif) for body text to improve readability in longer sections.
- Use a grid-based layout to organize content, mimicking the image shared. Keep design minimal and modern, to showcase the visual and investment information in a clean manner.
- Use a set of sharp and minimalist icons to visually represent each type of package, wallet feature, and action; and to support ease of use and rapid understanding.
- Subtle transitions and loading animations to enhance user engagement. Gentle animations when updating displayed amounts. Quick confirmation animations for successful wallet topups, withdrawals, and package purchases.
